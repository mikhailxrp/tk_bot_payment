## Правила проверки

- Проверяй только по коду и реальному выводу команд — не предполагай, что что-то работает
- Статически непроверяемое (реальная оплата, поведение в группе Telegram) → ⚠️ Требует ручной проверки
- Будь конкретным: файл и строка
- Не предлагай рефакторинг — только проверка DoD
- Любое нарушение разделов «Платежи / Робокасса» или «Панель: Server Actions и безопасность» — всегда Критическая проблема, даже если это одна строка
